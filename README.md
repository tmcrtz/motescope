# Motescope
Wii homebrew application for wiimote data measurement 

## Prerequesites
[devkitpro](https://devkitpro.org/wiki/Getting_Started) and install wii-dev (maybe gamecube-dev as well?)

## Compiling
1. Clone repo
    ```sh
    git clone https://github.com/tmcrtz/motescope.git
    ```
2. Set environment variables if needed
    ```sh
    DEVKITPRO=/opt/devkitpro
    DEVKITARM=/opt/devkitpro/devkitARM
    DEVKITPPC=/opt/devkitpro/devkitPPC
    ```
3. Enter project directory and `make`
4. Use `.dol` file
